# GPA Calculator 🎓

This project calculates GPA using:

* Global scale
* AOU scale

## Features

* Supports multiple grading systems
* Easy input
* Accurate GPA calculation

## How to use

Run the script:

```bash
python "GPA Calculator.py"
```

Enter number of courses: 3
